## Installation and build process

1. Clone this repository

```
git clone https://github.com/COS420-Fall24/TeamC.git
```

2. Navigate to the app folder

```
cd .\TeamC\comfort-read\
```

3. Install dependencies

```
npm install
```

4. Launch app

```
npm start
```
